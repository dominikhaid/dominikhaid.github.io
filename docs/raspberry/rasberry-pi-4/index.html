Raspberry Pi 4

The Raspberry Pi is a smale one plain computer, the cpu architektur is ARM based. The Rasberry can be used for all kinds of small projects, a NAS, ThinClient or as a Controller for all kinds of things are just some examples. I personally own a Raspberry Pi 4 with 8 GB of RAM and play around with it quite frequently. Below you will find some specifications and useful links.

Please see the offical page for a product offerview.
