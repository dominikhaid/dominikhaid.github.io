Software

Platform

Will are not running Pi Os on our Raspberry, instead we decided to use a standard Debian 11 ARM64 install, this will keep us as close as possible to our normal Debian 11 X64 environment we usually have.

Goals

We want to use our Pi as an Thin Client and Multimedia SharePoint. As for games we want to be able to use Google Stadia and RetroArch. For Multimedia we want basic apps like YouTube, Spotify, Facebook, Whatsapp and Plex as a media Center. The network will be setup for smb connection to connect our Nas and other Devices in our Network. Also onboard is Remmina for managing VNC connections to all of our Network clients and Vim/Nvim with an advanced config for text editing.

Description

The installation we use for our Raspberry 4 is based on our standard Debian Dev env. As prequirement an fresh Debian Bullseye install is needed. Wich can be easily done like on every other ARM machine using the official Debian 11 ARM iso, Wich can be found here. Below you will find a list of additional installed software.

Setup

Prequirement

  1. lnstall Debian 11 ARM on the Pi
  2. Clone the Pi Install repo
  3. cd pi-install
  4. sudo ./install
